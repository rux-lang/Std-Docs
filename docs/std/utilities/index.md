# Utilities

General-purpose utility modules.

## Import

```rux
import Std::Math;
import Std::Random;
import Std::Time;
```

## Description

The Utilities category groups general-purpose modules for math, random number generation, and time-related operations.

## Modules

| Module | Description |
| ------ | ----------- |
| [`Math`](math.md) | Mathematical functions and constants (planned). |
| [`Random`](random.md) | Pseudo-random number generation. |
| [`Time`](time.md) | Time-related utilities for sleeping, timing, and system time. |
